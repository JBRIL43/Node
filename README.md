# Task Manager Backend API

This is a Node.js Express backend for a Simple Task Manager System with authentication (JWT), user profile, and CRUD for tasks using MongoDB and Mongoose.

## Features

- User authentication (sign up, login) with JWT
- CORS enabled for all origins
- User profile endpoint
- Task CRUD endpoints (create, list with pagination/search, update status, delete)

## Tech Stack

- Node.js, Express.js
- MongoDB, Mongoose
- JWT, bcryptjs, dotenv

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your MongoDB URI and JWT secret
4. Run `npm start` (after adding the start script)

## Deployment

Deploy to Render, Railway, or similar. Submit the deployed link and GitHub repo as instructed.
